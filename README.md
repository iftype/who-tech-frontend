# frontend

우아한테크코스 크루 검색 서비스 프론트엔드. Next.js 15 App Router / TypeScript / Tailwind CSS v4.

## 빠른 시작

```bash
npm install
npm run dev    # http://localhost:3000
```

## 주요 명령어

```bash
npm run dev       # 개발 서버
npm run build     # 프로덕션 빌드
npm run lint:fix  # ESLint 자동 수정
```

## 환경변수 (.env.local)

```
NEXT_PUBLIC_API_URL=https://iftype.store
```
